# 🤿 Дневник погружений (Logbook)

Приложение для ведения дневника подводных погружений: добавление, просмотр, редактирование и анализ записей. Данные хранятся локально в браузере.

## 🎯 Возможности

- ✍️ **Добавление погружений** — дата, место, глубина, длительность, оценка, заметки; время начала необязательно (при пустом поле подставляется текущее)
- ✏️ **Редактирование** — в модальном окне, без его закрытия
- 📋 **Два режима** — «Редактирование» (форма + карточки) и «Просмотр» (таблица)
- 🔎 **Модальное окно** — детали погружения с анимацией открытия и закрытия
- 📊 **Статистика** — число погружений, макс. глубина, суммарное время под водой
- ✈️ **Таймер до полёта** — обратный отсчёт 24 часов с последнего погружения; карточка видна только пока не прошло 24 часа, снизу — дата и время окончания
- 🔍 **Поиск и сортировка** — в таблице по месту, глубине и времени, с пагинацией
- ⭐ **Рейтинг** — шкала от 1 до 5
- 💾 **localStorage** — данные сохраняются между сессиями

## ✨ Особенности реализации

- **Модульная архитектура** — компоненты сгруппированы по функциональности
- **Типы в отдельных файлах** — интерфейсы в `types.ts` каждой директории
- **Переиспользование** — `ActionButton`, `Stats`, `FormInput`, `Stars`
- **Валидация формы** — обязательны дата, место, глубина и длительность; время начала и заметки необязательны; будущие даты запрещены
- **Анимации модалки** — появление и исчезновение с учётом `prefers-reduced-motion`

## 📝 Формат данных

Каждое погружение (`Dive`):

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | `string` | Уникальный идентификатор |
| `date` | `string` | Дата (`YYYY-MM-DD`) |
| `time` | `string` | Время начала (`HH:MM`); если пусто при сохранении — текущее время |
| `site` | `string` | Место / сайт |
| `maxDepthM` | `number` | Максимальная глубина, м |
| `durationMin` | `number` | Длительность, мин |
| `notes` | `string` | Заметки, необязательно |
| `rating` | `number` | Оценка от 1 до 5 |
| `recordedAt` | `string` | Момент сохранения записи (ISO) |

### Таймер до полёта

- Берётся последнее погружение по дате и времени начала
- Если с него прошло меньше 24 часов — показывается карточка с обратным отсчётом и подписью «время до разрешенного полета»
- Снизу — дата и время окончания окна (`до …`)
- Если прошло 24 часа или погружений нет — карточка скрыта

Данные сохраняются в `localStorage` под ключом `dive-logbook`.
