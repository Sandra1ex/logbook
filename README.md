# 🤿 Дневник погружений (Logbook)

Приложение для ведения дневника подводных погружений с возможностью записи, просмотра, редактирования и анализа погружений.

## ✨ Особенности реализации

- **Модульная архитектура**: компоненты сгруппированы по функциональности
- **Переиспользование**: компоненты `ActionButton`, `Stats`, `FormInput` используются в разных частях приложения
- **Modal окно**: для просмотра деталей погружения без навигации
- **Валидация**: проверка дат (нельзя выбирать будущие даты)
- **Локальное хранение**: данные сохраняются в `localStorage` браузера

## 🎯 Возможности

- ✍️ Добавление погружений с детальной информацией (место, глубина, время, оценка, заметки)
- 📋 Просмотр истории в формате списка или таблицы
- ⭐ Рейтинг погружений по 5-звездочной шкале
- 📊 Статистика: общее количество, макс. глубина, суммарное время под водой
- 🔍 Сортировка по месту, глубине и времени
- 🕒 Дата со временем для точного отслеживания погружений
- 💾 Сохранение данных в браузере (localStorage)

## 📝 Формат данных

Каждое погружение содержит:
- `id` — уникальный идентификатор
- `date` — дата и время погружения (ISO)
- `site` — название места
- `maxDepthM` — макс. глубина в метрах
- `durationMin` — длительность в минутах
- `notes` — заметки
- `rating` — оценка (1-5 звезд)

Данные сохраняются в `localStorage` под ключом `dive-logbook`.
